## Quickstart

1. Install dependencies (use npm):

```bash
npm install
```

2. Start dev server (runs on port 4000):

```bash
npm run dev
```

3. Open http://localhost:4000

### Token Swap Form
- Select input token with icon
- Switch value between `FROM` value to `TO` value
- Error boundary

#### Error type
- Negative number
- Failed to load prices
- Swapping within same token
